<h2 align="center">About This Game</h2>

<p align="center">
  <b>MemorizeIT</b> is a memory and focus mini game originally created with HTML, CSS, and JavaScript.
</p>

<p align="center">
  This version has been redesigned and rebuilt using React to improve the project structure, user interface, responsiveness, and maintainability.
</p>

<p align="center">
  The game is designed to be simple to play, easy to understand, and suitable for desktop, tablet, and mobile devices.
</p>

<h2 align="center">Installation</h2>

<p align="center">
  Follow the steps below to run the project locally.
</p>

<h3>1. Clone the repository</h3>

```bash
git clone https://github.com/Rittiporn12/memorizeit.git
```

<h3>2. Go to the project folder</h3>

```bash
cd memorizeit
```

<h3>3. Install dependencies</h3>

```bash
npm install
```

<h3>4. Start the development server</h3>

```bash
npm run dev
```

<p>
  After running the command, open the local URL shown in the terminal.
</p>

```bash
http://localhost:5173/
```
